## 9. 对技术选型的影响

由于项目创建者不熟悉传统软件开发，因此技术选型应遵循：

1. 优先简单。
2. 优先文档丰富。
3. 优先生态成熟。
4. 优先容易让 Coding Agent 稳定生成代码。
5. 避免过早引入复杂工程化。
6. 避免微服务、复杂云架构和过度抽象。
7. 每个技术选择都需要解释原因、替代方案和风险。
