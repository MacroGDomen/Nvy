# 08 页面草图 / Page Wireframes

## 1. 文档目的

本文档记录 Nvy 第一版 MVP 的页面草图和 UI 风格要求。

当前仍处于文档阶段，不写业务代码，不搭建工程。本文档用于确认页面布局、视觉方向、交互状态和后续实现依据。

设计关键词：

- Google 风。
- Material Design 3 优先。
- 暗夜模式。
- 深灰背景。
- 淡暗紫色、淡暗粉色作为主强调色。
- 允许柔和渐变，但不能做廉价大面积高饱和渐变。
- 界面应有情绪和氛围，但仍保持资料库工具的可读性和可操作性。

### 2.2 色彩方向

推荐色彩语义，不是最终色值：

| 用途 | 方向 |
|---|---|
| 页面背景 | 深灰 / 近黑灰 |
| 面板背景 | 比页面背景略亮的暗灰 |
| 输入框背景 | 暗灰，类似 AI Studio 输入框，不使用白色 |
| 主强调色 | 淡暗紫 |
| 辅助强调色 | 淡暗粉 |
| 选中状态 | 淡紫或淡粉半透明底色 + 轻微发光 |
| Hover 状态 | 比普通状态略亮，带 tooltip |
| 文本主色 | 低刺眼白 / 浅灰 |
| 文本次色 | 中灰 |
| 危险操作 | 暗红或粉红偏危险色，但保持低饱和 |

### 2.3 形状与动效

- 使用 Material Design 3 的圆角、层级、状态反馈。
- 搜索框、对话框、下拉框、按钮使用圆角。
- 侧栏、顶部工具条和卡片不做厚重边框。
- 页面切换、侧栏展开、轮播、tooltip 和下拉栏使用短时平滑动效。
- 滚动内容接近固定顶部栏时使用渐隐遮罩，形成帘式滚动效果。
- 如果无法确定具体样式，优先采用 Material Design 3 暗色模式。

## 3. 全局布局

### 3.1 左侧状态栏

所有登录后的页面共用左侧状态栏。

折叠状态参考：

- `UI设计参考/GoogleGemini界面.png`
- `UI设计参考/A01.png`
- `UI设计参考/A02.png`
- `UI设计参考/A03.png`

展开状态参考：

- `UI设计参考/A04.png`

折叠状态规则：

- 左侧竖栏不要太宽，接近 Gemini 左侧图标状态栏。
- 默认只显示图标，不显示中文文字。
- 左上角显示 `NVY` logo 字样，风格参考 `UI设计参考/logo参考.md`，但实际不照抄大块 ASCII，只提取“粗、几何、品牌字样”的感觉。
- 首页、女优库、影片库、设置使用图标代表。
- 设置和用户头像固定放在底部。
- 当前选中的图标按钮需要有选中状态，参考 `A01.png` 的浅色提示底。
- 鼠标 hover 到按钮时显示 tooltip，参考 `A02.png`。
- 鼠标 hover 到 logo 时，logo 状态发生变化，参考 `A03.png`，提示可打开边栏。
- 点击 logo 或打开按钮展开侧栏。

展开状态规则：

- 展开后显示按钮中文名称和账户名，参考 `A04.png`。
- 展开后顶部仍显示品牌。
- 需要有一个按钮可以收起状态栏。
- 展开/收起都要有平滑动效。

建议图标语义：

| 页面 | 图标语义 |
|---|---|
| 首页 | 星火 / 起点 / 首页 |
| 女优库 | 人像 / 头像 |
| 影片库 | 影像 / 矩形播放卡 |
| 设置 | 齿轮 |
| 用户 | 头像 |

### 4.1 页面目标

首页不是普通资料面板，而是具有氛围的启动页：用户进入软件后，可以从轮播中的女优或影片开始，也可以在下方输入自然语言让大模型推荐。

### 4.2 布局草图

```text

│ NVY              │                                                         │
│ 首页 icon        │        女优头像轮播，整体沿向下凹的曲线排布              │
│ 女优库 icon      │       ○      ○        ○          ○        ○             │
│ 影片库 icon      │      名字    名字      名字        名字      名字          │
│                  │                                                         │
│                  │      影片封面轮播，整体也沿向下凹的曲线排布              │
│                  │    ▭       ▭          ▭            ▭       ▭             │
│                  │  番号 标题  番号 标题  番号 标题    番号 标题             │
│                  │                                                         │
│ 设置 icon        │                 {用户名}，今天你想从谁开始？              │
│ 用户头像         │                                                         │
│                  │        ┌───────────────────────────────────────┐        │
│                  │        │  Google AI Studio 风格首页对话框       │        │
│                  │        │  输入偏好、题材、标签或想看的类型       │        │
│                  │        └───────────────────────────────────────┘        │
└──────────────────┴─────────────────────────────────────────────────────────┘
```

### 4.3 轮播规则

- 女优头像和影片封面都参与首页轮播。
- 所有已记录女优和影片都参与轮播，加入软件即视为收藏。
- 女优头像下方显示默认名字。
- 影片封面下方显示 `番号 + 空格 + 名字`。
- 影片标题不要太长，超过影片封面长边宽度时使用 `......` 截断。
- 轮播整体呈向下凹的曲线，而不是水平直线列表。
- 轮播居中项需要比两侧项更大，越远离中央越小，形成视觉焦点。
- 每隔 1 秒自动向左轮播。
- 轮播到最左边的女优或影片，在下一秒渐隐并从左侧消失。
- 轮播项可点击，点击进入对应详情页。
- 轮播空状态需要有低调的新增入口，但不要破坏首页氛围。

### 4.4 首页对话框

- 对话框参考 Google AI Studio 底部输入框。
- 输入框背景为暗灰，不使用白色。
- 输入框圆角。
- 发送按钮、模型状态、配置缺失提示都保持低干扰。
- 配置有效时推荐默认可用。
- API Key 缺失时提示前往设置页配置。

### 7.1 页面目标

女优详情页需要保留资料编辑能力，但不能使用上一版 `字段名：__________` 的传统表单观感。字段输入使用参考 `K1.png`、`K2.png` 和 `K1K2源码.txt` 的浮动标签 / 圆角输入框方式，并按 Nvy 暗色风格调整颜色和尺寸。

```text

│                  │                                                         │
│                  │  个人评价输入区                                          │
│                  │                                                         │

│ 用户头像         │                                                         │
└──────────────────┴─────────────────────────────────────────────────────────┘
```

### 7.3 字段与按钮

- 除 `标签：` 和 `女优：` 保持 `XXX：` 这种标签式设计外，其他资料字段不使用 `XXX:__________`。
- 普通字段使用浮动标签输入框或圆角输入框。
- 输入框颜色、尺寸、圆角适配暗色 Google 风，不照抄参考源码的白底和原始尺寸。
- `添加标签`、`匹配`、`删除` 等按钮使用 `按钮源码.txt` 的交互动效方向，但颜色和大小根据整体风格重做。
- 删除按钮是危险操作，需低饱和危险色和二次确认。
- `删除` 不固定在右上角；只有滚动到详情页顶层时才能看到。

### 7.4 相关影片与滚动

- 下方相关影片一页显示 `3 * 3` 个封面。
- 封面下方只显示两行 `番号 + 空格 + 名字`。
- 超过两行使用 `......` 截断，每行不超过封面宽度。
- 鼠标滚轮或右侧滚动条滚动时，内容帘式滚动。
- 向上滚动的内容接近顶部时渐隐。

### 8.3 字段与按钮

- 影片详情页同样不使用传统 `字段名：__________` 形式。
- `标签：` 和 `女优：` 保持这种前缀设计，因为用户明确接受这两个区域。
- 普通字段使用浮动标签输入框或圆角输入框。
- `匹配`、`删除`、`添加标签`、`添加女优`、`重新翻译`、`取消翻译` 使用统一按钮语言。
- 作品类型下拉栏使用全局下拉栏样式。
- 删除影片需要二次确认。
- `删除` 不固定在右上角；只有滚动到详情页顶层时才能看到。

### 8.4 影评与时间戳

- 影评输入区要足够宽，不能像传统后台 textarea。
- 时间戳输入和添加按钮应靠近影评区域。
- 时间戳格式固定为 `[01:23:45]`。
- 插入时间戳时不打乱当前光标位置。

## 9. 设置页面

设置页用户未提供单独手绘草图，但仍沿用全局左侧状态栏和 Google / Material Design 3 暗色风格。

### 9.1 页面目标

设置页用于管理大模型配置、提示词、标签库、导入导出和安全说明。它应接近 Google AI Studio 的右侧设置面板气质：深色分区、紧凑但清楚、输入控件统一。

### 9.2 主要分区
